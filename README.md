# Wind Morrow 2

What Is This?
-------------

Some trash game I made.

How To Play
-----------------------

Launch topdownrpg.exe

Controls
----------
W, Up = Move up
<br />
S, Down = Move down
A, Left = Move left
D, Right = Move right
Escape = Close game
H = Use Health Potion
I = Open inventory
E = Interact
R = Reduce HP of Player
Menu only
-----------
Spacebar = Accept
